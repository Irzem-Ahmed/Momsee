# Momsee

Momsee is a clean, privacy-focused pregnancy tracking app built with Jetpack Compose. It helps you track your pregnancy progress, view key milestones, and see your weekly timeline—all without needing an internet connection.

## Features

- **Home Dashboard**: View exactly how many weeks and days you are along, with a visual progress bar and countdown to your due date.
- **Timeline**: A weekly breakdown of your pregnancy journey divided by trimesters.
- **Milestones**: Track significant developmental milestones (heartbeat, movements, viability, etc.) with dates calculated specifically for your pregnancy.
- **Privacy First**: No internet permissions, no trackers, and all data is stored locally on your device using Jetpack DataStore.
- **Dark Mode**: Supports system-wide dark mode with the option to manually override it in settings.
- **Modern UI**: Built using Material 3 and the Inter font family for a clean, accessible experience.

## Installation

You can build the app from source using Android Studio and Gradle.
